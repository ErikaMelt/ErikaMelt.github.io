Predicting Dyeing Process Outcomes

Using an XGBoost classifier, this study focuses on predicting dyeing process outcomes at Textil Olius. Key input parameters such as colorants, machine type, and weight are used to classify the success or failure of the process. To enhance interpretability, SHAP (SHapley Additive exPlanations) values are used.
